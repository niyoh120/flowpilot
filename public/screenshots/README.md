## How to Add Images

Since the images are hosted on an internal corporate network (docs.corp.kuaishou.com), they cannot be downloaded automatically. Please follow these steps:

1. Open share.md in your browser or editor
2. For each image URL, open it in your browser (if accessible)
3. Save the image to this directory with the filename specified above
4. Or take screenshots of the images from the rendered markdown

## Image Naming Convention

- `banner.png` - Main branding/header images
- `feature-*.png` - Feature demonstration screenshots
- `example-*.png` - Example diagram outputs
- `architecture.png` - Architecture diagram (already exists in /public)
- `example.png` - Main workspace screenshot (already exists in /public)

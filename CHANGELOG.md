# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-11-18

### Added
- Initial public release
- Conversational diagram generation with AI
- Real-time draw.io diagram preview
- Style replication from reference images
- Configurable drawing preferences (FlowPilot Brief)
- Multi-model comparison (up to 5 models)
- Support for multiple LLM providers:
  - OpenAI (GPT-4, GPT-3.5)
  - Anthropic Claude
  - Google Gemini
  - Amazon Bedrock
  - OpenRouter
- Multi-language support (English, Chinese)
- Diagram templates
- Various diagram types:
  - Sequence diagrams
  - Activity diagrams
  - Component diagrams
  - State machines
  - Deployment diagrams
  - Mind maps
  - Customer journey maps
  - Gantt charts
- PPT generation (Beta)
- Comprehensive documentation

### Changed
- Renamed project from "next-ai-draw-io" to "FlowPilot"
- Updated model provider naming for better generalization
- Improved environment variable configuration

### Removed
- Internal development documentation
- Company-specific references and configurations
- Unused legacy components
- IDE-specific configuration files

## [Unreleased]

### Planned
- More diagram templates
- Enhanced PPT generation
- Diagram export improvements
- Additional LLM provider integrations
- Plugin system for custom extensions
